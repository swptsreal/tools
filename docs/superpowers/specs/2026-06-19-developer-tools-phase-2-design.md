# Developer Tools Phase 2 Design

Goal: add Unit Converter, Number Base Converter, and Escape / Unescape as offline tools.

Scope:
- Unit Converter: length, weight, temperature, storage; from/to units; live result after Convert.
- Number Base Converter: input number + base, output binary/decimal/hex/octal/ASCII.
- Escape / Unescape: JSON string, JavaScript string, regex, URL component modes.

Constraints: no new dependencies, one tool per folder, Playwright E2E, update README roadmap.
